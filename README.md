# IT Helpdesk Operations Dashboard

## Problem Statement

IT support teams handle tickets across multiple departments, priorities, and support queues. Monitoring ticket distribution and operational workload through spreadsheets can be inefficient and time-consuming.

Managers require a centralized dashboard to track support activities and identify operational bottlenecks.

## Solution

Built an IT Helpdesk Operations Dashboard using Python for data preparation and Google Looker Studio for visualization.

The dashboard provides visibility into:

* Ticket volume
* Priority distribution
* Department workload
* Support queue performance
* Language-based ticket distribution

## Key Features

* Priority analysis
* Department-wise ticket tracking
* Queue monitoring
* Interactive filtering
* Operational KPI reporting
* Ticket distribution visualizations

## Skills Demonstrated

* Data Cleaning
* Python (Pandas)
* Business Intelligence
* Dashboard Development
* Operational Analytics
* Looker Studio

## Learning Outcome

This project strengthened my understanding of operational reporting, data preparation workflows, and dashboard design principles used in real-world support environments.
